## 前言
1.在git上新建一个repository
2.进入本地项目文件夹
```
git init  //把这个目录变成git可以管理的仓库
git add .  // 上传本文件夹下所有文件
git commit -m "first commit"   //把文件提交到仓库
git remote add origin 地址     //关联远程仓库
git push -u origin master   //把本地库的所有内容推送到远程库上
```
