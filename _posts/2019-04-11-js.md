## 前言


###  原型
+ 原型链的顶层是Object.prototype，而这个对象的是没有原型对象的。可在chrome的控制台里面输入：Object.__proto__输出是:function Empty() {}

### 引用类型

#### Object 类型
1. 创建Object实例的方式有两种：
第一种：使用new操作符后跟Object构造函数
```
var person = new Object();   与  var person = {};   相同
    person.name = 'niko';
    person.age = 20;
```
第二种：对象字面量表示法
```
var person = { 
 name : "Nicholas", 
 age : 29 
};
```
