### 文字两端对齐  `text-align-last: justify`


```
<div>
    <ul class="justify-text">
        <li>账号</li>
        <li>密码</li>
        <li>电子邮件</li>
        <li>通讯地址</li>
    </ul>
</div>

.justify-text li {
        margin-top: 5px;
        padding: 0 20px;
        width: 100px;
        height: 40px;
        background-color: #fee;
        line-height: 40px;
        text-align-last: justify;
        color: #2342ff;
    }
```
